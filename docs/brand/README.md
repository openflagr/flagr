# Flagr brand assets (option D)

Single approved identity: geometric **F monogram** + **FLAGR** wordmark.  
**PNG only** (no SVG). Light assets use **solid white** backgrounds; dark mark uses **solid black**.

## Source files

| File | Description |
|------|-------------|
| `flagr-logo.png` | Full lockup, black on white (docs header, OG/JSON-LD, README) |
| `flagr-mark.png` | Monogram only, black on white (light surfaces, org avatar) |
| `flagr-mark-on-dark.png` | Monogram only, white on black (UI navbar, dark surfaces) |
| `favicon.png` | 64×64 monogram, black on white |
| `apple-touch-icon.png` | 180×180 monogram, black on white |

## Installed copies

**Docs**

- `docs/public/images/logo.png` ← lockup
- `docs/public/images/logo-mark.png` ← `flagr-mark.png` (black on white)
